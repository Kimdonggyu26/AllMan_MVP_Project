## 📌 프로젝트 소개
MVP는 [영화 비교, 리뷰, 커뮤니티]를 제공하는 웹 프로젝트입니다.

## 🛠️ 사용 기술
- Front-end: HTML, CSS, JavaScript
- Back-end: Java, JSP, Servlet
- Database: Oracle DB
- 서버: Apache Tomcat 9

## ✨ 주요 기능
- 회원가입 / 로그인 / 로그아웃
- 영화 정보 CRUD / 리뷰 작성
- 댓글 / 좋아요 / 답글
- 취향분석 / 게시판

## 👤 역할
- 김동규 : 관리자 서비스 / 영화 정보 조회
  
           ▫️ 영화 관리 (등록, 수정, 삭제)
           ▫️ 영화 상세 정보 조회
           ▫️ 리뷰 작성 / 댓글
           ▫️ 댓글 좋아요
